01

引言

写在前面

本模板基于 Presentación UNAL， SYSU Beamer Template， THU Beamer Theme， 做出如下修改。

  • 去除背景的水印图片，背景水印图片会导致页面切换时产生屏闪现象。
  • 基于 Presentación UNAL 的排版风格，使用 SYSU Beamer Template 的颜色主题，采用 THU Beamer Theme 和 Presentación UNAL 的内容进行测试。
  • 本模板使用 MulanPSL-2.0 开源许可协议。

编译方法

本模板为纯 HTML/CSS/JS，无需编译，直接在浏览器中打开即可。

# 直接双击 index.html 打开浏览器即可浏览
# 导出 PDF：在浏览器中按 Ctrl+P，选择"另存为 PDF"
# 导出 PPTX：先用浏览器打印为 PDF，再转换为 PPTX
提示：推荐使用 Chrome 或 Edge 浏览器获得最佳效果。数学公式使用 KaTeX 渲染，代码高亮使用 Prism.js。

新功能：左文右图布局

自由排版

使用 CSS Grid/Flexbox 可以轻松实现各种布局组合，远比 LaTeX Beamer 灵活。

  • 左图右文 / 左文右图
  • 自定义比例分栏
  • 卡片式布局
  • 响应式适配

新功能：卡片布局

Block 区块

支持普通、警告、示例三种风格的 block 区块样式。

数学公式

使用 KaTeX 渲染 LaTeX 数学公式，速度快效果好。

代码高亮

基于 Prism.js，支持 Python、C、Java、HTML 等多种语言。

自由布局

CSS Grid 提供灵活的页面排版能力，不受 LaTeX 限制。

新功能：三栏分栏

第一栏
可以放置任意内容，包括文字、图片、列表等。
第二栏
每栏独立，互不影响，适合对比展示。
第三栏
三栏布局在学术论文中常用于方法对比。
