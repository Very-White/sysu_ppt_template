写在前面

本模板 fork 自 Nelson Cheung 的 SYSU HTML PPT 模板， 并在其基础上做了修改和增强。原始模板灵感来源于 Presentación UNAL、 SYSU Beamer Template 和 THU Beamer Theme。

  • 新增 PPTX 导出脚本（Playwright 截图 + python-pptx 生成），支持保留母版元素（进度条、水印等）。
  • 支持将幻灯片内容拆分到多个 HTML 文件，便于协作和版本管理。
  • 基于 Presentación UNAL 的排版风格，使用 SYSU Beamer Template 的颜色主题。
  • 本模板使用 MulanPSL-2.0 开源许可协议。

编译方法

本模板为纯 HTML/CSS/JS，无需编译，直接在浏览器中打开即可。

# 直接双击 index.html 打开浏览器即可浏览
# 导出 PDF：在浏览器中按 Ctrl+P，选择"另存为 PDF"
# 导出 PPTX：运行 python export_to_pptx.py
提示：推荐使用 Chrome 或 Edge 浏览器获得最佳效果。数学公式使用 KaTeX 渲染，代码高亮使用 Prism.js。
